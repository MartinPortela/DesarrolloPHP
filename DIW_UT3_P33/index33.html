3. Animación continua con @keyframes

Crea una animación continua (por ejemplo un icono que late o una burbuja que se desplaza) usando la propiedad animation.

La burbujita debe moverse o cambiar de tamaño de forma infinita mediante una animación CSS.

Los GIFs pueden ser utilizados para iconos, sin embargo, las animaciones CSS pesan mucho menos, por lo que la página se cargará más rápido para el usuario, no solo esto, si no que también son completamente controlables a diferencia de los GIFs. Para concluir, si la página necesita animaciones varias, es mejor utilizar animaciones CSS, sin embargo, para un icono, es bueno usar GIF, ya que son fáciles de utilizar y están habilitados en todos los navegadores.
